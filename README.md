# 📌 Overview

The **Hybrid Data Analytics System** is a full-stack AI-powered analytics platform designed for businesses, students, analysts, and non-technical users.

Instead of manually cleaning datasets, writing scripts, generating charts, and interpreting reports, users can simply upload a CSV file and instantly receive:

- Automated data cleaning
- Business intelligence insights
- Professional dashboards
- Retail analytics
- Forecasting predictions
- Interactive visualizations
- AI-generated recommendations

# ⚡ Installation & Setup

## 1. Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/hybrid-data-analytics-system.git
cd hybrid-data-analytics-system
```

---

## 2. Create Virtual Environment

```bash
python -m venv venv
```

### Windows
```bash
venv\Scripts\activate
```

### Linux / Mac
```bash
source venv/bin/activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4. Run Application

```bash
python app.py
```

---

## 5. Open Browser

```txt
http://127.0.0.1:5000
```
